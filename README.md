# Onboarding Service

A scalable and configurable onboarding platform built using Spring Boot.

The goal of this project is to create a generic onboarding system that can be used across multiple domains such as:

- Student Onboarding
- Employee Onboarding
- Voter Registration
- Citizen Registration
- Government Schemes Enrollment
- Customer Onboarding
- Vendor Onboarding
- Partner Onboarding

Instead of building separate onboarding solutions for each domain, this platform provides a common onboarding framework that can be configured based on business requirements.

## Features

- User Registration
- Authentication & Authorization
- JWT Based Security
- Role Based Access Control
- Configurable Onboarding Workflow
- Document Verification
- Status Tracking
- Audit Logging
- Centralized Exception Handling
- Standardized API Responses
- Database Migration using Flyway
- RESTful APIs

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Flyway
- JWT
- Maven

## Vision

To build a reusable onboarding platform that can onboard any entity (student, employee, voter, citizen, customer, vendor, etc.) through configurable workflows while maintaining security, scalability, and compliance.
